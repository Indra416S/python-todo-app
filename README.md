# Python To-Do List App

A simple command-line based To-Do List application built using Python.

## Features
- Add tasks
- View tasks
- Delete tasks
- Menu-driven interface

## Concepts Used
- Lists
- Functions
- Loops & Conditions
- User Input Handling

## How to Run
1. Download the file `todo_app.py`
2. Open terminal or command prompt
3. Run:
   python todo_app.py
